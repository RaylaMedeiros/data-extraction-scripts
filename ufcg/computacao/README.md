### Formato dos dados
```json
{
  "sala": "re-08",
  "disciplina": "aa3",
  "professor": "rohit",
  "categoria": "opt-especifica",
  "periodo (antigo;novo)": "*;*",
  "horario": [
    "t18",
    "i18" 
  ],
  "turma": "t1"
}
```

#### Legenda do atributo horário
  * s: Segunda 
  * t: Terça
  * q: Quarta
  * i: Quinta
  * x: Sexta
