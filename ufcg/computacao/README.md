## Modelo dos dados
```json
{
  "sala": "re-08",
  "disciplina": "aa3",
  "professor": "rohit",
  "categoria": "opt-especifica",
  "periodo (antigo;novo)": "*;*",
  "horario": [
    "t18",
    "i18" 
  ],
  "turma": "t1"
}
```

### Formato do horário: (dia)(horas)
A formatação do atributo horário está disposta da seguinte forma: (dias)(horas).  

Símbolo      | Legenda (dia da semana)
------------ | -----------------------
s            | Segunda 
t            | Terça
q            | Quarta
i            | Quinta
x            | Sexta

**Exemplo:** x14 (Sexta às 14 horas)
